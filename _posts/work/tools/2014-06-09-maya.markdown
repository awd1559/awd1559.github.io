# maya
## 创建物体

```
Create->NURBS->Primitive
Alt+左键		拖拽
Alt+右键		缩放
中键		移动
F 		使物体在屏幕居中
A 		屏幕上显示场景冲所有物体
```

```
F3 	多边形模式
F4 	surface模式
F5 	particle模式
```

```
4 线条显示
5 实体显示
6 
```

```
Y last Tool
G 重复上个命令
```

```
F8 Enter/Exit Component Mode
```

```
Window->Render Editors->Hypershade
	添加材质球
	鼠标中键拖动到物体上，赋予材质
```

```
Window - Animation Editor - Express Editor
	object.TranslateY = sin(time)
```

```
Skeleton->Joint Tool
	定义骨骼工具
```

```
Create Deforme->Cluster
	创建Cluster辅助对象
```
