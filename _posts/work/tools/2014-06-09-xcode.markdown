xcode

CMD+[     左缩进
CMD+]     右缩进


打开项目文件： ctrl+shift+o


command 控制Navigator
command 1-8 在左侧Navigator间跳转
command 0 显示／隐藏左侧Navigator



Alt 控制 右侧 Asset Utility editor

Command Alt 2-6 在右侧检测器间跳转
Command Alt 0    显示／隐藏右侧检测器

Command shift Y 显示／隐藏debugger

Command shift o 快速打开项目中的文件
