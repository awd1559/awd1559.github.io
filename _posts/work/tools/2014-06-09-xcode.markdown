# keymap
Standard Editor>Show Document Items 

------------------- | ---------------------- |
Open Quickly        | ctrl+shift+o           |
cmd + p             | function jump in file  |
cmd+shift+j         | Reveal in Project Navigator   |
⌘+⌃ +↓or↑           | *.m*.h切换  |
option + 左键        | show doc |
cmd + .             | 暂停模拟器 |
⌘+⌃ +←or→           | 前进后退文本文件 |
⌃+i                 | 自动排版代码 | 
⌘+[or]              | 左右缩进 | 
⌘+/                 | 注释 |
